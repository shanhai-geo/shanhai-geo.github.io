快速接入指南

1. 获取您的密钥

开通后您会收到：

  • API密钥（Key）：格式为 sh-xxxxxxxxxxxxxxxx
  • 接入地址（Base URL）：
https://api-proxy-daezqenetk.cn-beijing.fcapp.run/v1

2. 协议兼容

我们的接口完全兼容 OpenAI API 协议。如果您已经在使用 OpenAI 的代码，只需修改两个参数：

# 修改前（OpenAI官方） client = OpenAI(api_key="sk-xxx") # 修改后（山海智能） client = OpenAI( api_key="sh-你的密钥", base_url="https://api-proxy-daezqenetk.cn-beijing.fcapp.run/v1" )

3. Python 完整示例

from openai import OpenAI client = OpenAI( api_key="sh-你的密钥", base_url="https://api-proxy-daezqenetk.cn-beijing.fcapp.run/v1" ) # 调用对话 response = client.chat.completions.create( model="qwen-max", messages=[ {"role": "system", "content": "你是一个有用的助手"}, {"role": "user", "content": "帮我写一首诗"} ] ) print(response.choices[0].message.content)

4. cURL 示例

curl https://api-proxy-daezqenetk.cn-beijing.fcapp.run/v1/chat/completions \ -H "Authorization: Bearer sh-你的密钥" \ -H "Content-Type: application/json" \ -d '{ "model": "qwen-max", "messages": [{"role": "user", "content": "你好"}] }'

5. JavaScript / Node.js 示例

import OpenAI from 'openai'; const client = new OpenAI({ apiKey: 'sh-你的密钥', baseURL: 'https://api-proxy-daezqenetk.cn-beijing.fcapp.run/v1' }); const response = await client.chat.completions.create({ model: 'qwen-max', messages: [{ role: 'user', content: '你好' }] }); console.log(response.choices[0].message.content);
