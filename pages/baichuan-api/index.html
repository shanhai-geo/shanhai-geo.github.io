百川API接入要点

接入百川API需要在百川智能的开放平台进行注册和申请。API采用标准的HTTP接口格式，支持流式输出，开发者可以根据需求选择合适的调用方式。

百川API的接口设计与主流模型有一定差异，对于需要同时使用多个模型的团队，适配不同接口格式会增加开发工作量。
