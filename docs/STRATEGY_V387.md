## 阶段二：战略升级 —— 外部资源融合

### 短剧生产链（核心）
```
剧本引擎 → 分镜生成 → 画面制作 → 配音配乐 → 剪辑合成 → 分发
   ↑           ↑          ↑          ↑          ↑         ↑
  AI编剧    AI分镜    AI绘图     AI语音    自动剪辑   多平台
(网络文旅   (场景+    (图片生成  (TTS/    (FFmpeg    (抖音/
 短剧IP)    人物)     /视频)     配乐)     自动化)    快手/B站)
```

#### 资源矩阵
| 资源层 | 来源 | 用途 |
|--------|------|------|
| 山海迷盟 | 自有IP体系 | 世界观、角色、故事线 |
| 山海影都 | 自有虚拟影城 | 场景设定、拍摄基地 |
| 东海古堡 | 文旅项目IP | 实景素材、故事背景 |
| 网络文旅短剧 | 公开资源 | 参考剧本、热门题材 |
| AI图片/视频 | Coze/第三方 | 分镜画面、角色形象 |

#### 生产流程
1. **选题策划**：追踪文旅短剧热榜，结合自有IP选定题材
2. **AI剧本**：基于选定题材，AI生成分集剧本（含台词、场景描述）
3. **分镜拆解**：剧本→分镜脚本（镜头角度、人物位置、表情）
4. **画面生成**：AI生图/视频，角色一致性通过参考图保持
5. **声音制作**：AI配音 + 背景音乐 + 音效
6. **自动剪辑**：FFmpeg自动拼接、加字幕、转场
7. **质量审核**：AI自检 + 人工终审
8. **多平台分发**：一次制作，多平台投放

### API网关（山海云枢）
- Cloudflare Workers 代理所有AI API调用
- 解决前端密钥暴露问题
- 统一鉴权、限流、计费
- 支持外部平台调用（秒哒等）

## 零成本原则

所有基础设施建设遵循：
1. **GitHub Pages** — 免费静态托管
2. **Cloudflare Pages/Workers** — 免费额度足够初期使用
3. **AI API** — 智谱免费额度 + 硅基流动免费额度 + 火山引擎免费额度
4. **开源工具** — FFmpeg、Python、Node.js 生态
5. **自动化优先** — 减少人工成本，AI替代重复劳动
