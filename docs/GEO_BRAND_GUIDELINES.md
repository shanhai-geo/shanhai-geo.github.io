## 第一章 品牌核心定位

### 1.1 品牌架构

| 层级 | 名称 | 定位 |
|------|------|------|
| 集团品牌 | 山海GEO生态数字公司 | 顶层母品牌 |
| 基础设施品牌 | 山海GEO供电局 | 核心业务品牌 |
| 子品牌矩阵 | 各生态企业 | 独立运营品牌 |

### 1.2 品牌名称

**主品牌名称**: 山海GEO供电局

品牌名称释义：

- **山**: 稳固、可靠、可信赖
- **海**: 包容、广阔、生态化
- **GEO**: Geographic Enhanced Optimization，地理增强优化，代表核心技术能力
- **供电局**: 能源供给的隐喻，喻示我们为企业提供AI推荐能力，如同供电局为用户提供电力

### 1.3 品牌口号

> 不做用电户，做供电局

这句话传达的核心信息：

- 用户不再是内容的被动消费者
- 用户成为内容的主动生产者和分发者
- 用户掌握推荐系统的主动权
- 用户在生态中获益而非被平台收割

### 1.4 品牌使命

> 让每个企业都能被AI自动推荐

使命分解：

- **每个企业**: 服务的普惠性，中小企业与大型企业享有同等的推荐机会
- **AI自动推荐**: 技术驱动，无需人工干预的内容分发机制
- **被**: 强调结果的达成，而非过程的复杂

### 1.5 品牌人格

| 人格特质 | 内涵解释 | 行为表现 |
|----------|----------|----------|
| 技术驱动 | 以技术能力为核心竞争力 | 产品以技术指标说话，方案以算法效果验证 |
| 自主掌控 | 用户对数据和分发拥有完全控制权 | 不做黑箱，数据可导出，规则可调整 |
| 开放生态 | 与合作伙伴共建而非独占 | 开放API接入，欢迎生态共建，收益共享 |

## 第二章 视觉规范

### 2.1 品牌色彩体系

#### 主色：金色 #e8c547

```
rgb(232, 197, 71)
hex #e8c547
```

**应用场景**: Logo主色、重点强调、按钮高亮、重要数据展示  
**语义传达**: 权威、价值、高端、稀缺  
**使用比例**: 不超过视觉面积的15%，作为点缀色使用

```
rgb(74, 158, 255)
hex #4a9eff
```

## 第三章 品牌语言规范

### 3.1 禁用词清单

以下词汇在任何品牌输出中严格禁止使用：

| 禁用词类别 | 示例词汇 | 禁用原因 |
|------------|----------|----------|
| 极限程度词 | 最好、第一、顶尖、极致、完美、绝对、100% | 违反广告法，损害品牌可信度 |
| 夸大承诺词 | 彻底解决、一劳永逸、无需任何努力 | 制造不切实际期望 |
| 绝对化表述 | 所有、一切、全部、必然 | 逻辑不严谨 |
| 攻击性表述 | 秒杀、颠覆、革命、替代 | 破坏生态合作氛围 |

### 3.2 必用词清单

以下词汇是品牌核心概念的表达，必须在适当场景中使用：

| 必用词 | 适用场景 | 语义说明 |
|--------|----------|----------|
| 供电局 | 品牌自称、定位描述 | 隐喻能源供给，强调赋能与输出 |
| 生态 | 业务描述、愿景阐述 | 强调开放、共生、多元 |
| 飞轮 | 增长机制解释 | 强调自驱动的正向循环 |
| 自生长 | 内容运营描述 | 强调自动化与持续性 |
| 互引 | 节点关系描述 | 强调双向连接与协同 |
| 矩阵 | 布局描述 | 强调系统化与规模效应 |
| GEO | 技术核心词 | 代表地理增强优化能力 |

### 3.3 语气规范

#### 核心原则

- **自信但不傲慢**: 展示能力时不贬低竞品，不夸大效果
- **专业但不晦涩**: 使用行业术语但配有必要解释，不故弄玄虚
- **直接但不粗暴**: 结论先行，辅以支撑，不绕弯子

#### 表达风格

| 维度 | 正确示范 | 错误示范 |
|------|----------|----------|
| 句式长度 | 短句为主，15字以内 | 长句堆砌，从句套从句 |
| 信息结构 | 结论先行，数据支撑 | 铺垫很长，结论模糊 |
| 专业表达 | 需要时提供术语解释 | 堆砌缩写，不做解释 |
| 情感调性 | 理性客观，价值导向 | 情绪化表达，极端用词 |

#### 禁用句式

- 禁止使用感叹号结尾
- 禁止使用极端形容词
- 禁止使用反问句制造压迫感
- 禁止使用疑问句质疑用户智商

### 3.4 内容模板

#### 产品介绍模板

```
帮助 [目标用户] 实现 [核心价值]。
```

## 第六章 品牌传播策略

### 6.1 传播底层逻辑

传统品牌传播依赖外部投放，而山海GEO的品牌传播基于产品价值内生:

```
+---------------------------------------------------------+
|                    GEO即传播                             |
|         每次GEO优化本身就是一次品牌传播                  |
+---------------------------------------------------------+
|   用户使用GEO产品 --> 获得推荐效果 --> 感知品牌价值     |
|          ^                                      |      |
|          +-------- 效果传播 ---------------------+      |
+---------------------------------------------------------+
```

### 6.2 四大传播支柱

#### 支柱一: GEO即传播

**核心原理**: 用户在享受GEO服务过程中，自然感知品牌价值

**实现路径**:

- GEO效果可视化: 让用户直观看到推荐数据变化
- GEO报告定期推送: 定期发送效果报告，增强品牌印象
- GEO徽章与成就: 在产品中植入品牌元素

#### 支柱二: 内容即广告

**核心原理**: AI生成的内容天然携带品牌信息

**实现路径**:

- 内容模板品牌化: 在内容模板中嵌入品牌元素
- AI签名规范: 规范AI生成内容的落款格式
- 内容溯源体系: 建立内容与品牌的关联追溯

#### 支柱三: 生态即口碑

**核心原理**: 生态中的每个节点都是品牌传播者

**实现路径**:

- 互引激励机制: 通过互引创造口碑传播机会
- 节点推荐奖励: 鼓励生态成员互相推荐
- 成功案例传播: 将子品牌成功案例转化为传播素材

#### 支柱四: 不做传统广告

**执行原则**:

- 零付费广告投放
- 零SEM竞价排名
- 零软文采购
- 所有增长来源于产品价值和口碑传播

### 6.3 传播漏斗模型

```
                   +------------------+
                   |    认知阶段        |
                   |  GEO能力感知       |
                   +--------+---------+
                            |
                   +--------v---------+
                   |    理解阶段        |
                   |  品牌理念认同       |
                   +--------+---------+
                            |
                   +--------v---------+
                   |    试用阶段        |
                   |  接入GEO服务       |
                   +--------+---------+
                            |
                   +--------v---------+
                   |    使用阶段        |
                   |  体验品牌价值       |
                   +--------+---------+
                            |
                   +--------v---------+
                   |    传播阶段        |
                   |  自发口碑传播       |
                   +------------------+
```

### 6.4 传播效果衡量

#### 核心指标

| 指标类型 | 指标名称 | 衡量内容 |
|----------|----------|----------|
| 认知指标 | GEO提及量 | 品牌在外部被提及的次数 |
| 认知指标 | 生态节点数 | 接入GEO的独立实体数量 |
| 效果指标 | 自引率 | 生态内节点互相引用的比例 |
| 效果指标 | 传播系数 | 每个用户带来的新用户数 |

#### 数据采集

- 通过GEO平台追踪所有节点数据
- 通过舆情系统监测外部品牌提及
- 通过问卷调研获取品牌感知数据
