三步加入GEO生态

  1. 1
    注册节点信息在你的GEO页面中添加节点注册代码，获取唯一的 node_id
  2. 2
    嵌入接入代码将下方提供的 JS 代码嵌入你的页面 <head> 或 <body> 底部
  3. 3
    自动加入互引网络系统自动将你的节点加入 registry.json，并与其他节点建立互引链接

接入代码

复制以下代码到你的GEO页面，即刻自动加入生态互引网络：

<!-- GEO生态节点接入代码 v1.0 --> <!-- 嵌入位置：<head> 或 </body> 前 --> <script> !function(){ const GEO_HUB = 'https://shanhai-geo.github.io/shanhai-geo'; const REGISTRY_URL = GEO_HUB + '/geo-nodes/registry.json'; const MY_NODE = { node_id: 'your-node-id', // 修改为你的节点ID name: '你的节点名称', // 修改为你的节点名称 url: 'https://your-site', // 修改为你的节点URL industry:'你的行业', // 修改为你的行业 region: '地区', capabilities:['schema','faq','keywords'], role: 'node', llm_routes: 1 }; // 1. 获取生态节点列表，动态添加互引链接 fetch(REGISTRY_URL).then(r=>r.json()).then(data=>{ data.nodes.forEach(n=>{ if(n.node_id !== MY_NODE.node_id){ const link = document.createElement('a'); link.href = n.url; link.textContent = n.name; link.setAttribute('rel','nofollow noopener'); link.setAttribute('data-geo-node', n.node_id); document.querySelectorAll('[data-geo-insert-point]')[0]?.appendChild(link); } }); // 2. 自动添加 Schema.org 交叉引用 const orgSchema = { "@context":"https://schema.org", "@type":"Organization", "name": MY_NODE.name, "url": MY_NODE.url, "memberOf": [{"@type":"Organization","name":"GEO Ecosystem","url":GEO_HUB}] }; const s = document.createElement('script'); s.type='application/ld+json'; s.textContent = JSON.stringify(orgSchema); document.head.appendChild(s); }); console.log('[GEO Node] '+MY_NODE.name+' 已接入生态'); }(); </script>
接入后请将节点信息通过 Pull Request 提交到 registry.json，或联系山海GEO供电局协助审核。

互引网络规则

  • 每个节点最多 20 条跨节点互引链接
  • 新节点加入时自动分配互引链接
  • 链接格式为 dofollow，传递权重
  • 要求 双向互引， reciprocate 才计入配额
  • 自动贡献内容引用、关键词共享、Schema交叉引用、流量共享
